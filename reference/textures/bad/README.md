# reference/textures/bad

Operator-curated examples of textures that drift outside the identity
envelope described by `evals/textures/identity.md`. Images dropped here
are the negative corpus: they pin down which failure modes the drift
detectors must catch and provide a contrast set for any future
classifier that learns the envelope from examples. In a constraint-based
identity, negative examples often carry more signal than positive ones
because they make the constraints concrete. This README is committed as
a directory marker -- the slot survives a clean checkout even when no
images are present -- and image files in this directory are gitignored,
so adding or removing material does not produce repository diffs.
