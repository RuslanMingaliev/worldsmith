# reference/textures/good

Operator-curated examples of textures that fall inside the identity
envelope described by `evals/textures/identity.md`. Images dropped here
are the positive corpus: they define what "in-style" means for the
project, supply the data that numeric thresholds in the constitution are
calibrated against, and serve as targets for the future generator. This
README is committed as a directory marker -- the slot survives a clean
checkout even when no images are present -- and image files in this
directory are gitignored, so adding or removing curated material does
not produce repository diffs.
